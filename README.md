## Running locally

[Official docs](https://docs.github.com/en/pages/setting-up-a-github-pages-site-with-jekyll/testing-your-github-pages-site-locally-with-jekyll)

1. `bundle install`
2. `bundle exec jekyll serve`
3. Go to http://localhost:4000.
